✅ Build and deploy GH Pages / build (push) Successful in 23s

Finally, this is my own personal website, built using a static site generator. I think this is exactly what I need because I have tried many website-building services—from scratch, no-code, low-code, and pre-built solutions—but none of them felt right. They all had too many features I didn't need or want.

I built this website in just a few hours during my break sessions while learning.

This website is truly "static" because I created it using Zola. Whenever I want to write a new post, I simply create a new Markdown file and write whatever I want. You can do the same—just visit their website and start building your own.

The biggest challenge I faced was deploying it to the internet using GitHub Actions and setting up a custom domain. It might not be difficult for others, but as a beginner in DevOps, it was quite a challenge for me. So when I finally saw the "successful" message, I felt really happy!

This website will serve as a documentation of my learning journey. Currently, I'm exploring game development, alongside iOS development. Game development seems like a fun and refreshing activity for me.

My English and writing are still a work in progress, so my posts may be a little unpolished, but I'm imporving step my step!
